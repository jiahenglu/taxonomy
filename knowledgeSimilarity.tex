\begin{abstract}

A string similarity join finds all similar pairs between two input string collections. It is an essential operation in
many applications, such as data integration, data cleansing and record linkage. This paper investigates a novel challenge to integrate taxonomy with string similarity joins. In principle, the taxonomy presents a general-purpose strategy to improve the accuracy of string joins by enriching data with semantics-based knowledge  (i.e., a set of IS-A hierarchies).

In this paper, we first propose several new functions to measure the semantic similarity between two strings based on the taxonomy knowledge. We show three interesting properties our new measurements guarantee. Then given two collections of strings, we describe new algorithms to perform the efficient string similarity joins based on the proposed functions.  The technique relies on the insight that two strings are similar if they share certain number of common ancestors in the taxonomy tree. We optimize our algorithms by  building a cost model and an adaptive strategy to select the best parameters to speedup the performance.  The extensive performance evaluation shows that our algorithms can utilize the taxonomy to improve the effectiveness of string joins  and  scale very well with important parameters like data size and taxonomy size.


\end{abstract}
